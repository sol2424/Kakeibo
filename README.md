# 家計簿アプリ製作

1.はじめに

Java Silver SE17取得の能力を示す為と自分で普段使いすることを目的に
GUIで動く家計簿アプリの制作を行う。

（progateなどの学習の記録とかも書く？）

仕様技術

・Java SE17

・javafx-sdk-17.0.15

・JavaFX Scene Builder 23.0.1


・ChatGPT
